# Service Provider Frontend

React frontend for the Service Provider platform.

## Features

- User authentication (Login/Signup)
- Service browsing and search
- Shopping cart
- Booking management
- User profile management
- Provider dashboard
- Responsive design

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will run on `http://localhost:5173`

## Build

```bash
npm run build
```

## Technologies

- React 19
- Vite
- React Router DOM
- Framer Motion
- React Icons
- Axios
